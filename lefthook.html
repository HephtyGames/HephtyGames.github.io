Left Hook Game

This is not a HephtyGames Original but I was responsible for programming the core game mechanics and integrating the game with WebGL to ensure it runs smoothly on various platforms, including the web. This involved designing efficient game logic, implementing interactive gameplay elements, and ensuring that the physics and controls were responsive across different environments. Additionally, I handled the WebGL integration, optimizing the game for web performance and managing the challenges of running a Unity-based game in a browser environment. The entire process required a focus on gameplay quality and technical optimization to deliver a seamless player experience.
